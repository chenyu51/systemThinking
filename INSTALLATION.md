# 📦 SystemCanvas - 安装指南

## 前置要求
- Chrome 浏览器 (版本 90 或更高)
- 文本编辑器 (可选，用于开发)

## 安装方式

### 方式 1：开发者模式加载（推荐用于开发）

1. **打开 Chrome 扩展页面**
   - 在 Chrome 地址栏输入：`chrome://extensions/`
   - 或者：菜单 → 更多工具 → 扩展程序

2. **启用开发者模式**
   - 点击页面右上角的"开发者模式"开关

3. **加载项目**
   - 点击左上角"加载已解压的扩展程序"
   - 选择本项目的根目录 (`/Users/yuchen/Documents/my-github/canvas`)
   - 等待加载完成

4. **验证安装**
   - 在 Chrome 工具栏中应该看到 SystemCanvas 图标
   - 点击图标应该看到快速菜单

### 方式 2：脚本自动安装（仅限 Mac 用户）

```bash
# 进入项目目录
cd /Users/yuchen/Documents/my-github/canvas

# 运行安装脚本
chmod +x install.sh
./install.sh
```

## 首次使用

1. **打开插件**
   - 点击 Chrome 工具栏中的 SystemCanvas 图标

2. **选择操作**
   - **新建画布**: 创建新的系统图
   - **打开最近的画布**: 恢复上次保存的画布
   - **选择模板**: 从预设模板开始

3. **开始绘图**
   - 在左侧工具栏选择工具（节点、连线等）
   - 在中间画布区域绘制
   - 在右侧面板编辑属性

## 常见问题

### Q: 插件加载后不显示？
A: 
1. 确保启用了"开发者模式"
2. 检查是否加载了正确的目录
3. 刷新扩展程序页面（F5）
4. 重启 Chrome 浏览器

### Q: 画布加载很慢？
A: 
1. 这是首次加载，正常情况下 1-2 秒
2. 如果超过 5 秒，可能是浏览器性能问题
3. 尝试关闭其他扩展程序

### Q: 数据保存在哪里？
A: 
1. 所有数据存储在 Chrome 本地存储 (localStorage)
2. 支持手动导出为 JSON 文件备份
3. 导入 JSON 可恢复之前的画布

### Q: 能否在多台电脑间同步？
A: 
1. MVP 版本仅支持本地存储
2. 可通过导出 JSON 然后手动导入来转移
3. v2.0+ 版本会支持云同步

## 开发者指南

### 修改插件

1. **编辑任意源文件后**
   - 在扩展程序页面找到 SystemCanvas
   - 点击"重新加载"按钮（或按 Ctrl+Shift+R）

2. **调试**
   - 在扩展程序页面点击"背景页面"查看后台日志
   - 在画布页面右键 → 检查，使用开发者工具调试

3. **检查错误**
   - 打开画布页面 → 右键 → 检查
   - 在控制台查看错误信息

## 卸载方式

1. 打开 Chrome 扩展页面 (`chrome://extensions/`)
2. 找到 SystemCanvas
3. 点击"移除"按钮
4. 确认删除
