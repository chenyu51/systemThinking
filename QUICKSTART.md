# 🚀 SystemCanvas - 快速开始指南

## 5分钟上手

### 第1步：安装 (1分钟)

1. 打开 Chrome，输入 `chrome://extensions/`
2. 右上角启用"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择项目目录

✅ 完成！你应该在工具栏中看到 SystemCanvas 图标

---

## 第2步：打开插件 (1分钟)

1. 点击工具栏的 SystemCanvas 图标
2. 选择"✚ 新建画布"
3. 等待主画布加载

![快速入门](./assets/quickstart.png)

---

## 第3步：绘制第一个系统图 (3分钟)

### 创建节点

```
1. 左侧"⭕ 添加节点" → 选中工具
2. 在画布中心点击 → 创建节点
3. 重复步骤2创建第二个节点
```

**提示**: 你会看到两个蓝色矩形节点。

### 编辑节点标签

```
1. 右侧面板中的"节点标签"输入框
2. 输入你的标签，例如 "市场需求"
3. 按回车或点击其他地方保存
```

### 连接节点

```
1. 左侧"➜ 绘制连线" → 选中工具
2. 从第一个节点拖拽到第二个节点
3. 松开鼠标完成连接
```

**你现在有了一个简单的因果关系！**

### 标记反馈类型

```
1. 在画布中点击连线（会高亮显示）
2. 右侧面板选择反馈类型：
   • 正反馈 (+) - 绿色 ✓
   • 负反馈 (-) - 红色 ✓
   • 中立 - 灰色
```

## 示例：简单的产品反馈循环

### 场景描述
分析：**产品做得越好 → 用户满意度增加 → 口碑传播 → 新用户增加 → 产品需求增大 → 投入更多资源 → 产品做得越好（循环）**

### 绘制步骤

1. **创建5个节点**：
   - 产品质量
   - 用户满意度
   - 口碑传播
   - 新用户数
   - 开发投入

2. **排列节点**（左侧到右侧的流程）

3. **连接成循环**：
   - 产品质量 → 用户满意度 (+)
   - 用户满意度 → 口碑传播 (+)
   - 口碑传播 → 新用户数 (+)
   - 新用户数 → 开发投入 (+)
   - 开发投入 → 产品质量 (+)

4. **观察结果**：
   - 整个循环都是正反馈 (+)
   - 这是一个"增强回路" (reinforcing loop)
   - 表示一旦启动会加速增长

## 保存和导出

### 自动保存
- 每次编辑后，按 `Ctrl/⌘ + S` 保存
- 数据存储在浏览器本地

### 导出为文件

```
工具栏 → [导出▼] 选择格式：
├─ PNG    (用于演示、分享)
├─ SVG    (用于专业编辑)
└─ JSON   (用于备份、导入)
```

### 恢复之前的画布

```
工具栏 → [打开] → 选择导出的 JSON 文件
```

## 实用技巧 💡

### Tip 1: 使用颜色区分
- 同类节点用相同颜色
- 不同阶段用不同颜色
- 例如：绿=增长、红=风险、蓝=基础

### Tip 2: 添加节点描述
- 在右侧面板添加详细说明
- 帮助理解复杂概念
- 便于后续回顾

### Tip 3: 分层绘制
1. 先画关键节点（核心流程）
2. 再添加反馈回路
3. 最后整理布局和样式

### Tip 4: 标记延迟关系
- 某些影响需要时间才能显现
- 在连线上勾选"有延迟 (//)"
- 虚线表示延迟的因果关系

### Tip 5: 批量整理
1. 使用 Ctrl+A 选择全部（开发中）
2. 右对齐、居中对齐
3. 使用网格吸附整理布局

## 故障排除 🔧

### 问题 1: 节点无法移动
**解决**: 
1. 确保左侧选择的是"👆 选择工具"
2. 不是"✚ 添加节点"或"➜ 绘制连线"

### 问题 2: 连线无法连接
**解决**:
1. 确保选择了"➜ 绘制连线"工具
2. 从一个节点拖拽到另一个节点（不是同一个）
3. 确保两个节点都存在

### 问题 3: 修改后没有保存
**解决**:
1. 按 `Ctrl/⌘ + S` 手动保存
2. 工具栏点击"💾 保存"按钮
3. 状态栏应显示"画布已保存 ✓"

### 问题 4: 导出的PNG太小
**解决**:
1. 先缩小到合适大小
2. 导出PNG
3. 或者导出SVG，用其他工具编辑
