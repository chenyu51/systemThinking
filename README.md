# 📊 SystemCanvas - Chrome插件需求文档实现

一款轻量级系统思维与可视化分析工具的 Chrome 插件。帮助用户快速绘制系统循环图、因果回路图、思维导图。

## 📁 项目结构

```
SystemCanvas/
├── manifest.json              # Chrome 插件配置文件 (Manifest V3)
├── background.js              # 后台服务 Worker
├── popup/
│   ├── popup.html            # 插件弹窗UI
│   └── popup.js              # 弹窗交互逻辑
├── panel/
│   ├── index.html            # 主画布界面
│   ├── canvas.js             # 核心画布逻辑（SVG渲染、交互）
│   ├── panel.js              # 工具栏菜单交互
│   ├── store.js              # 数据存储层 (Chrome Storage API)
│   ├── node.js               # 节点数据模型 & SVG生成
│   └── edge.js               # 连线数据模型 & SVG生成
└── assets/                   # 图标文件夹（待建）
    ├── icon-16.png
    ├── icon-48.png
    └── icon-128.png
```

## ✨ 核心功能（MVP v0.1）

### 基础绘图功能
- ✅ **节点创建** - 添加文本节点，可自定义标签内容
- ✅ **连线绘制** - 节点间绘制带箭头的连线，表示因果关系
- ✅ **连线类型** - 支持正反馈(+) / 负反馈(-) / 中立
- ✅ **画布操作** - 拖拽移动节点、平移视图
- ✅ **样式设置** - 节点颜色、形状（矩形/圆形/菱形）、连线颜色

### 辅助功能
- ✅ **本地保存** - 将图表保存为 JSON 到浏览器本地存储
- ✅ **导出功能** - 支持导出为 PNG/SVG/JSON 格式
- ✅ **快捷键** - 
  - `Delete` 删除选中元素
  - `Ctrl/⌘ + Z` 撤销
  - `Ctrl/⌘ + Y` 重做  
  - `Ctrl/⌘ + S` 保存

## 🚀 快速开始

### 安装和加载

1. **本地开发加载**：
   - 打开 Chrome 浏览器
   - 输入 `chrome://extensions/`
   - 启用"开发者模式"（右上角）
   - 点击"加载已解压的扩展程序"
   - 选择本项目目录

2. **打开插件**：
   - 点击工具栏中的插件图标
   - 选择"+ 新建画布"或"打开最近的画布"

### 使用流程

#### 创建系统图
1. 选择**"添加节点"**工具
2. 在画布上点击放置节点
3. 双击节点编辑文字标签
4. 在右侧面板调整颜色、形状

#### 建立关系
1. 选择**"绘制连线"**工具
2. 从节点A拖拽到节点B
3. 在右侧面板选择反馈类型（正/负/中立）
4. 添加连线标签

#### 保存和导出
1. **保存**: `Ctrl/⌘ + S` 或点击工具栏"保存"按钮
2. **导出**: 点击"导出 ▼" 选择格式
   - PNG: 用于演示和分享
   - SVG: 用于专业编辑
   - JSON: 用于备份和导入

## 🔧 技术栈

- **前端框架**: 原生 JavaScript（轻量、无依赖）
- **图形引擎**: SVG + JavaScript（原生实现）
- **存储**: Chrome Storage API（本地存储）
- **兼容性**: Chrome 90+ (Manifest V3)
- **体积**: < 2MB（轻量级）

## 📋 核心代码说明

### store.js - 数据存储层
- 封装 Chrome Storage API
- 提供节点、连线的增删改查
- 支持 JSON 导入导出

### node.js - 节点模型
- 定义节点数据结构
- 生成 SVG 元素
- 支持不同形状和类型的渲染

### edge.js - 连线模型
- 定义连线数据结构
- 生成 SVG 路径和箭头
- 支持正负反馈类型标记

### canvas.js - 核心画布
- SVG 渲染引擎
- 节点/连线的创建、选择、编辑
- 拖拽、缩放、平移交互
- 撤销/重做功能
- 快捷键绑定

### panel.js - UI交互
- 工具栏按钮事件
- 导出功能实现
- 菜单交互逻辑
